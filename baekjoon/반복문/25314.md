## 🧩 풀이 흐름

1. 입력값 `N` (4의 배수, 4 ≤ N ≤ 1000) 받기
2. `N`을 4로 나눈 몫만큼 "long" 반복
3. 마지막에 "int" 추가하여 완성

---

## 🧾 내 코드

<details>
<summary>✅ 시도한 코드 (런타임: 92ms)</summary>

```node
const fs = require('fs');
const input = fs.readFileSync(0, 'utf-8').trim();

console.log(Array(Math.ceil(input / 4)).fill('long').join(' ') + ' int');
```
</details>
