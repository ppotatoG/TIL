# join()

>## [join](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/join)

>join() 메서드는 배열의 모든 요소를 연결해 하나의 문자열로 만듭니다.

```js
const arr = ['여자아이들', '전소연', '사랑해'];

console.log(arr.join()); // 여자아이들,전소연,사랑해
console.log(arr.join(' ')); // 여자아이들 전소연 사랑해
console.log(arr.join('♥')); // 여자아이들♥전소연♥사랑해
```
